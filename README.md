# Strip Metadata

This is a script with exiftool that remove **ALL** meta from the current directory (were the script is place).

You must have install *exiftool*:

`sudo pacman -S exiftool` | `brew install exiftool` | `sudo apt install libimage-exiftool-perl`

## Scripts

### view_meta.sh
You see all meta data from ALL images.

### stripmeta.sh
You ERASE **ALL** metadata of all images.


## Run

```
> ./view_meta.sh
> ./stripmeta.sh
```
